## 為什麼要寫部落格？

寫部落格是一個很好的學習方式。透過將所學整理成文字，可以幫助自己更深入地理解知識，同時也能與他人分享經驗。

## 技術選擇

這個部落格使用了以下技術：

- **Astro** - 現代靜態網站生成器，效能優異
- **Tailwind CSS** - 實用優先的 CSS 框架
- **Markdown/MDX** - 內容撰寫格式
- **Cloudflare Pages** - 部署平台

### 為什麼選擇 Astro？

Astro 的「零 JavaScript」理念非常吸引我。它只在需要互動的地方才加載 JavaScript，使得頁面載入速度極快。

```javascript
// Astro 元件範例
---
const greeting = "Hello, World!";
---

<h1>{greeting}</h1>
```
